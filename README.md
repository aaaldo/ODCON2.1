# ODCON2.1

This is an updated version of the original ODCON2, based on the original ODCON2 build.

![Finished ODCON build](odcon-ibi.jpeg?raw=true)

Compared to the original, this version includes those parts modified by the DIYLightGund Discord community : 

- Body - Front - Modified.stl : added extra clearance on screws

- Body - Grip - Modified.stl : added extra space near the trigger for thicker cables or USB plugs

- Frame - Lower - Modified V1.2.stl : added extra space in the electronics cavity

- Retainer - 4mm.stl : alternate version for 4mm wide springs

- Trigger - 4mm.stl : alternate version for 4mm wide springs

- Body - Spring Stop - 4mm.stl : alternate version for 4mm wide springs
